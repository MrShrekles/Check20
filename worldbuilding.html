Lock Types
  • Mundane — Physical key; pick or force
  • Blood — Requires specific blood
  • Soul — Bound to a soul
  • Rune — Detonates when tampered
  • Mimic — Sentient; requires password
  • Illusory — Hides secondary trap
  • Tech — Keypad / biometric
  • Magic — Basic incantation
  • Celestial — Star key
  • Chaos — Illogical entropy key
  • Chronomancy — Specific moment only
  • Crystal — Resonance match
  • Dragon — Scale, blood, or sigil
  • Elemental — Primal element
  • Nature — Sacred leaf or token
  • Vozian — Infernal pact
Trap Types
  • Dart — Toxic / Piercing
  • Rune — Any damage type
  • Trap Door — Impact
  • Gas — Toxic / Eclipse
  • Net — Pinned condition
  • Illusion — Confused / Fear
  • Explosive — Fire / Thunder
PL Scaling
  • Damage: 1d6 per PL
  • Range: 5ft + 5ft per PL
  • +Table every 3 PL
  • Disarm: Physical or Mental check
