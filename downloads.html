How to install in Roll20
  1. In your Roll20 game, go to Game Settings → Character Sheet Template and choose Custom.
  2. Paste the contents of ARC20 Charsheet v8.html into the HTML tab.
  3. Paste the contents of ARC20 Style v8.css into the CSS tab. Save.
  4. For the API script, go to API Scripts and create a new script. Paste the contents of ARC20 API v8.js and save.
  5. Use !importmonster from the Monster Book on this site to send monsters directly to Roll20.
